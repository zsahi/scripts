# multipleurlopener.html
We have many urls that we need to explore manually in browser one by one. How to keep track of all URLs we have visited and which urls are pending. This html page help to organize and stay focused by highlight color current url under investigation, pending urls to explore. Also we can open multiple URLs with single click just like chrome extension 'Open Multiple URLs'
